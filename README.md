**Executive Virtual Assistant · Systems, Content & Operations Support for Busy Founders & Professionals.**

This is a modern, responsive portfolio website designed to showcase Stella Igwe's executive virtual assistant services, experience, tools, and client work. It includes interactive elements, smooth animations, and social media preview cards for effective online presence.

🔗 **Live URL:** [https://IamSalvation.github.io/stella_portfolio/](https://IamSalvation.github.io/stella_portfolio/)

---

## ✨ Features

- **Dynamic Hero Section** – Typewriter effect, floating particles, and a rotating image frame.
- **20+ Mastered Tools** – Displays a comprehensive toolkit (Notion, Trello, Canva, ChatGPT, etc.).
- **Services & Experience** – Clear breakdown of offered services and professional timeline.
- **Case Study Modals** – Interactive cards that open detailed project breakdowns (Notion, Skincare, Travel, Google Workspace).
- **Auto-Smooth Carousels** – Seamless, continuous sliding carousels for:
  - **Work Samples** (9 unique project images)
  - **Testimonials** (10 client reviews)
- **Visual Enhancements**:
  - Floating particles & geometric shapes for depth.
  - Custom cursor & glow trail (desktop).
  - Noise texture overlay for a premium feel.
  - 3D tilt effect on cards.
- **Social Media Preview (Open Graph / Twitter Cards)** – Shares a rich preview with your profile photo, title, and description on LinkedIn, X, Facebook, and WhatsApp.
- **Fully Responsive** – Optimised for desktop, tablet, and mobile devices.
- **Contact Form** – Integrated with Formspree (`https://formspree.io/f/meeyeole`) for easy messaging.

## 🚀 Getting Started (Local Development)

Follow these steps to run the project locally:

1. **Clone the repository**
   ```bash
   git clone https://github.com/IamSalvation/stella_portfolio.git
   cd stella_portfolio
Add your images
Place your profile photo and work sample images into the assets/ folder.
(Make sure the filenames match those referenced in index.html).

Open the site
You can simply open index.html in your browser. For the best experience (to avoid CORS issues with fonts or icons), use a local development server:

Using VS Code – install the Live Server extension and click "Go Live".

Using Python – python -m http.server 8000 and visit http://localhost:8000.

🌐 Deployment (GitHub Pages)
This repository is configured for GitHub Pages. To deploy your live site:

Push your code to the main branch.

Go to your repository on GitHub → Settings → Pages.

Under Branch, select main and save.

After a few minutes, your site will be live at:
https://IamSalvation.github.io/stella_portfolio/

📝 Customisation Tips
Change the profile image – Replace assets/stella.jpg with your own photo.

Update social links – The contact section already includes your LinkedIn, Instagram, X, and WhatsApp links. Update the href attributes in index.html if needed.

Modify the typewriter phrases – Edit the phrases array in script.js (currently: Seamless Operations, Strategic Support, Business Growth, Workflow Efficiency, Peace of Mind).

Adjust the carousel speed – Look for 3500 (work samples) and 4000 (testimonials) in script.js to change the slide interval.

Update the Formspree endpoint – The form currently submits to https://formspree.io/f/meeyeole. Replace it with your own Formspree ID if needed.
